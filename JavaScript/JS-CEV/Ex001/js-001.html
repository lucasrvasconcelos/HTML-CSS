Declaração de variáveis

var nome = window.prompt('Digite seu nome')

Para mostrar na tela

window.alert('Bem vindo, ' + nome)

Para declarar uma variável como real(Float)

var n1 = Number.parseFloat(window.prompt('Primeiro valor'))

Para declarar uma variável como Inteiro(Int)

var n1 = Number.parseInt(window.prompt('Primeiro valor'))

Pode deixar também o próprio js decidir dependendo dos dados recebidos

var n1 = Number(window.prompt('Primeiro valor'))

Para transformar uma variável em string

String(n) ou n.toString()

window.alert('Sua média é ' + String(s))

Template string

var nome = window.prompt('Nome: ')
var idade = Number.parseInt(window.prompt('Idade: '))
window.alert(`Seu nome é ${nome} e sua idade é ${idade}`)

Contar quantas letras tem a variável

variavel.length

Transformar toda a variável em maiúsculo

variavel.toUpperCase()

Transformar toda a variável em minúsculo

variavel.toLowerCase()

Para escrever no html se usa:

document.write(`Texto exemplo ${variavel}`)

obs: Entreo do document.write pode se usar tags html

Para fixar casas decimais

document.write(`O número formatado fica ${variavel.toFixed(4)}

Para substituir o ponto por virgula

document.write(`O número formatado fica ${num.toFixed(2).replace('.' , ',')}`)

Para transformar um valor real em reais

document.write(`O número formatado fica ${num.toLocaleString('pt-BR',{style: 'currency', currency:'BRL'})}`)
