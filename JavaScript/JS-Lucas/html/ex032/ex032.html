Formulários

Form: tag referênte ao formulário

Autocomplete: Eu digo se deve ultilizar os valores armazenados no cache do navegador. Usando na tag form só tem a opções de on e off. Usando nas tags de imputs, você especifica qual tipo de dado é pra trazer no auto complete.(autocomplete="username" Por exemplo)

Label: Onde eu referencio uma tag de texto a aquele input

Input: É a tag onde iremos especificar qual tipo de formulário iremos pedir

Action: É onde eu digo qual ação aquele formulário vai ter

Method: Eu digo o método do formulário. Post ou Get

Get: Mostra os dados na URL

Post: Não mostra os dados na URL

Placeholder: Mostrar algum valor no funco da caixa

min-lenght: Valor mínimo para ser digitado

max-lenght: Valor máximo para ser digitado

min: Valor mínimo para números

max: Valor máximo para números

size: Quantidade de letras para serem mostradas

Required: Para dizer que aquele campo é obrigatório

step: Onde eu informo de quanto em quanto meu numero vai pular, ultilizado no input do tipo number

Value: eu digo um valor padrão no imput, caso eu já queira que o formulário já selecione uma data específica por exemplo

Checked: Já deixa o checkbox ou radio marcado

option: É para eu colocar as opções disponíveis em em conjunto com as tags select ou datalist
