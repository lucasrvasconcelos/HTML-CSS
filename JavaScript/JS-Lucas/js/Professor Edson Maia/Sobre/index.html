Curso de Java Script (Front/Back)

Características do Java Script

1-Todo Navegador Web tem um interpretador front-end do JS.

2-V8 é o interpretador JavaScript, também chamado de máquina virtual JS.

3-Os códigos podem se embutidos no HTML ou pnkados em arquivos externos.

4-Para usar o Js fora dos avegadores, é necessário instalar o Node JS.

Exemplo Tag JS

<script>
("Olá, mundo")
</script>

Link externo

<script src="arquivo.js"> </script>
